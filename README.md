# CoinCollector
金币收集小游戏

## 项目简介
一款简单的休闲收集小游戏，玩家操控角色移动，收集场景内的金币。

## 开发语言
Lua

## 游戏功能
- 角色键盘移动控制
- 金币拾取与统计
- 保存玩家拾取金币数目
- 简单游戏场景布局

## 项目运行
1. 克隆本仓库到本地
```bash
git clone https://github.com/eighteen231018/CoinCollector.git
```
## 注意
想要能够保存金币数需要保存到roblox然后打开Experience Setting中的Security，找到Enable Studio Access to API Services
